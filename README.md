# 🛒 Black Friday Sales Analysis using Python & Power BI

## 📌 Project Overview
This project analyzes Black Friday retail sales data to uncover customer purchasing behavior, revenue trends, demographic insights, and strategic business opportunities using Python and Power BI.

The project demonstrates an end-to-end analytics workflow including:
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Statistical Testing
- Customer Segmentation
- Dashboard Development
- Business Recommendations

---

## 🎯 Business Objective
The objective of this project is to:
- Analyze customer purchasing patterns
- Identify high-value customers
- Understand demographic spending behavior
- Discover top-performing product categories
- Generate data-driven business recommendations

## 📂 Project Workflow

### 1️⃣ Data Cleaning & Preprocessing
- Handled missing values in product category columns
- Standardized city tenure values
- Converted data types for consistency
- Verified dataset integrity

### 2️⃣ Feature Engineering
Created advanced business metrics such as:
- Customer Lifetime Value (CLV)
- Category Breadth
- Average Purchase Value
- City Loyalty Score
- Customer Segmentation

### 3️⃣ Exploratory Data Analysis (EDA)
Performed analysis on:
- Customer demographics
- Gender-based purchasing behavior
- Age-wise spending patterns
- City-wise revenue contribution
- Product category performance

### 4️⃣ Statistical Analysis
- Conducted T-Test for gender-based spending analysis
- Built correlation heatmaps for feature relationships

### 5️⃣ Dashboard Development
Developed an interactive Power BI dashboard featuring:
- Executive Overview
- Revenue Analysis
- Customer Insights
- Product Performance
- Strategic Recommendations

## 📊 Key Insights

### 📍 Revenue & Geography
- City Category B contributed the highest revenue share
- Mid-tier cities showed strong purchasing activity

### 👥 Customer Behavior
- Customers aged 35–55 demonstrated the highest spending capacity
- Male customers showed higher average purchase values
- High-value customers contributed disproportionately to revenue

### 🛍️ Product Insights
- Product Categories 5, 1, and 8 generated the highest transaction volume
- Customers purchasing across multiple categories spent more on average

### 📈 Customer Segmentation
- High-value customers are critical for long-term revenue
- One-time buyers require targeted retention strategies
- Cross-category engagement positively impacts customer spending

---

## 💡 Strategic Recommendations
- Focus premium campaigns on high-spending age groups
- Strengthen retention strategies for high-value customers
- Increase cross-selling and bundled offers
- Use localized campaigns in underperforming regions
- Introduce budget-friendly products for younger demographics
